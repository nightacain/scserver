# Shadow Checkers 2

Игра в шашки с уникальными механиками и визуальными эффектами.

## Особенности
- Современный пользовательский интерфейс
- Звуковые эффекты
- Система магазина
- RPG элементы

## Установка и запуск
1. Клонируйте репозиторий
2. Откройте `index.html` в вашем браузере

## Технологии
- HTML5
- CSS3
- JavaScript
- Работа со звуком
- Анимации
